# Aspectra - API

## Prerequisites

- [Node.js](https://nodejs.org/en/) - JavaScript runtime built on Chrome's V8 JavaScript engine.
- [pnpm](https://pnpm.js.org/) - Fast, disk space efficient package manager.

### Initial steps

**Install modules**

```
pnpm install
```

**Prepare database**

```
pnpm db@dev
```

**Run in dev mode**

```
pnpm dev
```

**or production**

```
pnpm start
```
